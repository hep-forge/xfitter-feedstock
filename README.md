[![Build & Upload](https://github.com/hep-forge/xfitter-feedstock/actions/workflows/autoupload.yml/badge.svg)](https://github.com/hep-forge/xfitter-feedstock/actions/workflows/autoupload.yml)

[![Anaconda Platforms](https://anaconda.org/hep-forge/xfitter/badges/platforms.svg)](https://anaconda.org/hep-forge/xfitter)

Feedstock for [xfitter](https://gitlab.cern.ch/fitters/xfitter) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

xFitter - A PDF fit program from HERA.

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `2.2.1` |
| linux-arm64 (`linux-aarch64`) | ❌ not published |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge xfitter
```
